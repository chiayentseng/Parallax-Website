視差捲動效果

作者：Vineet Kumar

什麼是視差捲動？

視差捲動是一種網頁設計技巧，讓背景的移動速度比前景慢。當訪客向下捲動網站時會產生 3D 效果， 增加空間層次感並創造更沉浸的瀏覽體驗。視差效果來自視覺錯覺：人眼會把近處物體看得比遠處更大， 因而讓遠處物體看起來移動得更慢。這種錯覺長期被運用在各種媒介中，以營造更逼真的效果。 它最早出現在傳統動畫中，例如迪士尼的《白雪公主與七個小矮人》，以及像《超級瑪利歐》這類的遊戲。 隨著 CSS 與 HTML 的進步，視差效果逐漸演進成為今日網頁設計的重要手法。

2007 年，網頁開發者 Glutnix 在部落格中分享了早期的瀏覽器視差捲動實作，包含使用 JavaScript 與 CSS 2 的範例程式碼與示範，支援當時的 Internet Explorer 6 等瀏覽器。不過直到 2011 年， HTML5 與 CSS 3 普及後，這項技術才真正受到網頁設計師的歡迎。支持者認為它是擁抱網路流動性的簡單方式， 也常用來提升互動性與整體體驗。不過普渡大學在 2013 年的研究指出：「…雖然視差捲動提升了部分使用者體驗， 但未必能改善整體的使用者體驗。」


為什麼要使用視差圖像？

    設計師嘗試視差圖像主要有兩個原因：

  • 有助於說故事

    視差圖像非常適合用來敘事。好的設計重在溝通，當我們設計網站時， 希望與訪客交流，並講述關於產品、服務或品牌的動人故事。 視差捲動能讓訪客沉浸在獨特的線上體驗中，甚至可以透過單頁網站 呈現完整故事，無需在多個頁面間切換。

  • 提升使用者參與感

    視差圖像能提升使用者參與度。降低跳出率是網頁設計的重要目標之一， 跳出率指的是只看了一頁就按返回或關閉分頁的訪客比例。高跳出率會影響搜尋排名， 因此降低跳出率的努力都是值得的。精心製作的視差頁面能帶來動態體驗， 讓使用者停留更久。


如果你想在網站中加入視差捲動效果，可以參考以下建議：

  • 選擇性使用

    視差捲動在特定元素上使用時更有力量，而不是整個網站每一頁都使用。 你可以把它放在頁首或標題，或只用在首頁。目標是強化設計， 而不是分散使用者注意力。

  • 影像壓縮

    視差捲動會使用多種媒體檔案與 CSS 形狀來營造深度。 避免使用過大的圖片檔造成載入過慢，影響效能與體驗。

  • 運用色彩

    視差捲動不只關乎前景與背景的圖片，也包含色彩運用。 背景可以使用較低飽和度，前景使用更鮮明的色彩， 以強化層次感。
